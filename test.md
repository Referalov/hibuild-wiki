# Тестовая страница

Это страница для проверки работы вики. Здесь можно тестировать разные форматы текста.

## Оформление текста

Здесь можно использовать **жирный текст**, *курсив* или ~~зачеркнутый~~.

### Списки

Маркированный список:
- Первый элемент
- Второй элемент
  - Вложенный элемент
- Третий элемент

Нумерованный список:
1. Раз
2. Два
3. Три
